# File Organizer（授業課題・ファイル自動整理スクリプト）

このスクリプトは、特定のフォルダ内に散らばったテキストファイルを、その「作成日時」に基づいて自動的にリネームし、年・月ごとのフォルダへ分類・整理するPythonツールです。

## 概要

本ツールは、授業内のPython演習にて、講師作成のテキスト教材を参考にしながら作成した課題作品です。

## 学習目的

- Pythonの標準ライブラリ（os / shutil / datetime / glob）の理解
- ファイル操作・ディレクトリ操作の基礎習得
- 実務を想定した自動化スクリプトの設計経験


## 主な機能

1.  **自動リネーム**
    - ファイルの作成日時（誕生時間）を取得し、`YYYY年M月D日_ファイル名` という形式に自動で書き換えます。
2.  **フォルダの自動生成**
    - 整理先のフォルダ（`interview/20XX年/X月`）が存在しない場合、自動的にディレクトリを作成します。
3.  **自動仕分け（移動）**
    - リネームしたファイルを、該当する年月フォルダへ自動的に移動（整理）します。

## フォルダ構成の変更例

実行前、`memo/` 内にあったファイルが、実行後には日付情報が付与され、階層化された `interview/` フォルダへ移動します。

### 実行前
```text
.
├── memo/
│   ├── 松嶋菜々子.txt
│   └── 平手友梨奈.txt
└── file.ipynb
```

### 実行後
```text
.
├── interview/
│   ├── 2020年/
│   │   └── 3月/
│   │       └── 2020年3月16日_平手友梨奈.txt
│   └── 2021年/
│       └── 1月/
│           └── 2021年1月8日_松嶋菜々子.txt
└── file.ipynb
```

## 使い方

1.  整理したいテキストファイルを `memo/` ディレクトリの中に配置します。
2.  `file.ipynb` のセルを実行します。
3.  実行後、`interview/` フォルダ内に日付ごとに整理された状態で保存されます。

## 必要ライブラリ

このスクリプトはPythonの標準ライブラリのみを使用しています。

* `os`
* `shutil`
* `datetime`
* `glob`

## 実行環境

- Python 3.9.6
- Jupyter Notebook
